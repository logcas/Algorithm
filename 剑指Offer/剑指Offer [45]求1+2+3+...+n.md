# 剑指Offer [45]求1+2+3+...+n

标签（空格分隔）： 算法

---

求1+2+3+...+n，要求不能使用乘除法、for、while、if、else、switch、case等关键字及条件判断语句（A?B:C）。

## Solution
关键是想到利用&&运算符去制造短路，达到if判断的效果。

```javascript
function Sum_Solution(n)
{
    // write code here
    // 利用 && 短路模拟 if
    let sum = n;
    n > 0 && (sum += Sum_Solution(--n));
    return sum;
}
```
